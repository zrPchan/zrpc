# このフォルダについて

このフォルダは、PythonのFlaskを使って自力で簡単なWebアプリケーションを作成するためのものです。
手作り（handmade）で行うことを目指しています。
AIはヒントやアドバイスを提供するだけで、コードの自動生成は行いません。
